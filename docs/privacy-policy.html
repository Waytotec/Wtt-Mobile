WTT Control 개인정보처리방침

최종 업데이트: 2025년 12월 31일

요약: WTT Control 앱은 개인정보를 수집하지 않습니다. 모든 데이터는 사용자의 기기에서만 처리되며, 외부 서버로 전송되지 않습니다.

1. 수집하는 개인정보

WTT Control은 어떠한 개인정보도 수집하지 않습니다.

  • 사용자 계정 정보를 요구하지 않습니다
  • 위치 정보를 수집하지 않습니다
  • 기기 식별자를 수집하지 않습니다
  • 사용 통계나 분석 데이터를 수집하지 않습니다

2. 앱의 기능 및 데이터 처리

WTT Control은 로컬 네트워크 내의 IP 카메라를 관리하기 위한 앱입니다:

  • 카메라 검색: 로컬 네트워크에서 UDP 브로드캐스트를 통해 카메라를 검색합니다
  • 카메라 설정: 카메라의 IP 주소, 서브넷 마스크, 게이트웨이를 설정합니다
  • 카메라 재부팅: 네트워크를 통해 카메라에 재부팅 명령을 전송합니다
  • 웹 설정 페이지: 카메라의 웹 인터페이스에 접근합니다

모든 통신은 로컬 네트워크 내에서만 이루어지며, 인터넷을 통해 외부 서버로 데이터가 전송되지 않습니다.

3. 네트워크 권한

앱은 다음 권한을 사용합니다:

  • INTERNET: 로컬 네트워크 카메라와 통신
  • ACCESS_NETWORK_STATE: 네트워크 연결 상태 확인
  • CHANGE_NETWORK_STATE: USB-C LAN 어댑터 사용 시 네트워크 바인딩

이러한 권한은 오직 로컬 네트워크 내 카메라 제어 목적으로만 사용됩니다.

4. 데이터 저장

앱은 영구적인 데이터를 저장하지 않습니다. 검색된 카메라 목록은 앱 실행 중에만 메모리에 유지되며, 앱 종료 시 삭제됩니다.

5. 제3자 서비스

WTT Control은 제3자 분석 도구, 광고 네트워크, 또는 외부 서비스를 사용하지 않습니다.

6. 아동 개인정보

이 앱은 산업용 카메라 관리 도구로서 아동을 대상으로 하지 않으며, 의도적으로 13세 미만 아동의 개인정보를 수집하지 않습니다.

7. 개인정보처리방침 변경

본 개인정보처리방침이 변경될 경우, 이 페이지를 통해 공지합니다. 중요한 변경 사항은 앱 업데이트 시 알림을 통해 안내할 수 있습니다.
